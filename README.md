# Yrgotchi

A tamagotchi with a "web dev student @ Yrgo" theme. Built using Vite, React and PixiJS.
URL to demo: [Yrgotchi](https://yrgotchi.netlify.app)

# Installation

1. Clone the repo
2. npm i
3. npm run dev
